# Discord Bot Privacy Policy

This Privacy Policy explains how the **State of the Core Bot** (“Bot”) collects, uses, and protects data when it operates in a Discord server.

## 1. What We Collect
The Bot only collects public GitHub repository URLs that appear in messages within servers it is added to. No other content or personal information (usernames, IP addresses, etc.) is collected or stored.

## 2. How We Use the Data
- The extracted URLs are saved locally on the server where the Bot runs.
- The data is used solely for the purpose of providing a list of GitHub repositories mentioned in that Discord community.

## 3. Data Sharing & Third Parties
The Bot does **not** share collected URLs with any third parties, nor does it transmit them outside the server’s environment. The bot's owner may frequently generate charts, graphs, and reports of the current State of the Meshcore development ecosystem in a manually generated manner, specifically curated for public consumption on occasion, but is not obligated to do so. 

## 4. Retention
URLs are retained until the Bot is removed from the server or the data file (`github_urls.txt`) is deleted by the Bot owner or server administrator.

## 5. Your Rights
- **Access & Deletion**: Server owners or administrators can delete the Bot and its stored data at any time.
- **Opt‑Out**: The Bot cannot be opted out of on a per‑message basis; however, removing the Bot from the server stops all future scraping.

## 6. Security
The Bot stores URLs in plain text. It is up to the Bot’s host to secure the file system and prevent unauthorized access.

## 7. Compliance
This Bot complies with Discord’s Terms of Service, GDPR (if applicable), and CCPA. By using the Bot, you confirm that your server’s activities comply with all relevant laws. The bot stores no personally identifiable information.

## 8. Changes to This Policy
We may update this policy from time to time. Updated versions will be posted in the repository and are effective immediately upon posting.

## 9. Contact
If you have questions about how we handle data, please contact the Bot’s maintainer at **<dan@wi7fi.dev>**.
